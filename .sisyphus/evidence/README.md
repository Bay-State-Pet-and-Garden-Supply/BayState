# QA Evidence Directory

This directory contains QA artifacts, screenshots, and verification outputs collected during development tasks.

## Purpose

- Store verification screenshots and outputs
- Document test results and validation evidence
- Provide traceability for QA processes

## Contents

Files in this directory are ephemeral and should not be committed to version control.
Only this README and .gitkeep are preserved.

## Usage

Evidence files are automatically generated during task execution and verification steps.
